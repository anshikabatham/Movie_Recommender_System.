# 🎬 Movie Recommender System

A Machine Learning-based web application that suggests movies based on user preferences. Built to understand content-based filtering and model deployment.

## 🚀 Technical Stack
* **Language:** Python
* **Machine Learning:** Scikit-Learn (Cosine Similarity)
* **Web Framework:** Streamlit
* **Data Processing:** Pandas

## 🧠 How It Works
1. **Data Processing:** Cleaned and combined movie features (genres, keywords, cast, crew) using Pandas.
2. **Vectorization:** Converted text data into numerical vectors to find patterns.
3. **Similarity Score:** Calculated the Cosine Similarity between movies to find the closest matches.
4. **Interactive UI:** Deployed a clean, user-friendly web interface using Streamlit where users can select a movie and instantly get top 5 recommendations.
